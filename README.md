# DecodeLabs Project 2 - Data Classification Using AI

## 📌 Project Overview
This project uses Machine Learning to classify data using the Iris dataset.  
It trains a Decision Tree model and predicts flower types based on input data.

---

## 📁 File Information
- File Name: project2.py
- File Type: Python (.py)

---

## 🚀 How to Run
Install required library:
pip install scikit-learn

Run the program:
python project2.py

---

## ⚙️ Features
- Uses real dataset (Iris dataset)
- Splits data into training and testing sets
- Trains Decision Tree classifier
- Makes predictions on test data
- Calculates accuracy score

---

## 🧠 Concept Used
- Machine Learning (Supervised Learning)
- Classification Algorithm
- Decision Tree Model
- Train/Test Split
- Model Evaluation
